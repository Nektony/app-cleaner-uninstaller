# Changelog     

  All notable changes to App Cleaner & Uninstaller will be documented in this file.

  ## [9.1] - February 2026

  ### Added
  - Security Insights: view app permissions for system settings and folders, check
  Apple notarization status
  - AI App Summary to help you understand what each app on your Mac is for
  - Detection of available updates for iOS apps installed on Mac

  ### Improved
  - Updates tab with an expanded database of supported apps
  - Memory usage and overall performance

  ### Fixed
  - Minor bugs and stability issues

  ## [9.0.4] - January 2026

  ### Improved
  - App update mechanism, including Homebrew and App Store apps
  - Expanded the list of apps that can be updated
  - Mechanism that matches files to their corresponding apps for removal
  - Detection and cleanup of application support files
  - Various stability and reliability improvements

  ## [9.0.2] - November 2025

  ### Improved
  - Updating of installed applications

  ## [9.0.1] - November 2025

  ### Improved
  - User interface for better compatibility with the latest version of macOS

  ## [9.0] - October 2025

  ### Added
  - Complete app redesign for a cleaner, more intuitive experience

  ### Improved
  - Faster scanning process for even better performance
  - Expanded the list of supported apps for updates
  - Mechanism that matches files to their corresponding apps for removal
